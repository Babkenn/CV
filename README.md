## Babken Pokrikyan – One‑Page React CV

This is a single‑page React CV with a modern dark UI, glassmorphism, and subtle animations.

### Run locally

1. Install dependencies:

```bash
npm install
```

2. Start the dev server:

```bash
npm run dev
```

3. Open the URL printed in the terminal (usually `http://localhost:5173`).

### Build for production

```bash
npm run build
npm run preview
```
